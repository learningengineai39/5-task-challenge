# 5 Task Challenge

This repository contains 5 frontend mini-projects built using **HTML, CSS, and JavaScript**.

## 📂 Projects Included

📁 Folder: `sudoku-game`

A Sudoku puzzle game where users solve number-based logic puzzles.

### Features:

* Interactive puzzle board
* Number validation
